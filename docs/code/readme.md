# Como utilizar

Execute estes comandos na pasta raiz do projeto.

### Instalação

```bash
$ npm install
```

### Executar

```bash
# desenvolvimento
$ npm run start

# modo watch
$ npm run start:dev

# modo produção
npm run start:prod
```

## Testar

```bash
# testes unitários
$ npm run test

# testes e2e
$ npm run test:e2e

# testes de cobertura
$ npm run test:cov
```
